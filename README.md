# Craft-lytra
Wait, That's Elytra Craftable?! (เดี๋ยวนะ ปีกมันคราฟต์ได้ด้วยเหรอ?!)

"ปีกมันคราฟต์ไม่ได้! อย่าดูช่องอื่นให้มาก"
แพ็กนี้สร้างมาเพื่อพี่แท็ก Tackle4826 โดยเฉพาะ เพราะวันนี้... ปีกมันคราฟต์ได้แล้วโว้ยยย! (ถ้าใส่ส่วนเสริมนี้)

🪽 ฟีเจอร์หลัก
เพิ่มสูตรคราฟต์ปีก Elytra เข้ามา (ใช่ อย่างเดียวเลย)

เล่นเวอร์ชันไหนก็จัดไป: มีให้โหลดทั้งแบบ Datapack (Java Edition) และ Add-on (Bedrock Edition) ไม่ว่าจะสายคอมหรือสายมือถือก็ปีกคราฟต์ได้กันทั้งน้านน!!!

วิธีการคราฟต์: ใช้ Phantom membrane x4, Leather x4, Nether Star x1 
 <br> ![Elytra Crafting Recipes](https://github.com/MineGarp/Craft-lytra/blob/main/images/elytra_crafting_recipe.png)

📦 วิธีติดตั้ง
เลือกติดตั้งตามเวอร์ชันที่คุณเล่นได้เลย:

☕ สาย Java (Datapack)

โหลดไฟล์ไปโยนใส่โฟลเดอร์ datapacks ของโลกคุณ

พิมพ์คำสั่ง /reload หรือเข้าโลกใหม่

📱 สาย Bedrock (Add-on)

โหลดไฟล์ .mcpack แล้วกดเปิดไฟล์เพื่อติดตั้งเข้าเกม

ไปที่ตั้งค่าโลก แล้วกดเปิดใช้งานในหน้า Behavior Packs
